# Dominion Card Game - Text-Based Implementation

A Python implementation of the popular deck-building card game **Dominion**, playable as a single-player experience against AI opponents or in automated matches.

## About Dominion

Dominion is a deck-building card game where players start with a small deck of basic cards and gradually acquire more powerful cards throughout the game. The goal is to accumulate the most Victory Points by game end, while managing your deck's efficiency and strategic purchases.

## Features

- **Text-based interface** - Pure Python implementation with no GUI dependencies
- **Deck-building mechanics** - Acquire cards, manage your deck, and optimize your strategy
- **Multiple card types** - Support for Treasure, Victory, Curse, and Action cards
- **AI opponents** - Play against computer-controlled players with strategic decision-making
- **Customizable gameplay** - Configure game rules, card sets, and player counts
- **Single-player focus** - Practice against AI opponents or run automated matches
